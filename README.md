## Physical machine

You don't even need to clone this repo.

Install Debian (or Raspbian etc.) in your machine.

Login and become root.

Then run:

```
bash -c "$(wget -O - https://raw.githubusercontent.com/vemarsas/wiedii-bootstrap/main/bootstrap.sh)"
```
